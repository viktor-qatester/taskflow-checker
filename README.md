# 🚀 TaskFlow Checker

**TaskFlow Checker** — это Fullstack пет-проект таск-менеджера, разработанный специально для практики QA-инженеров. В приложение встроен специальный симулятор багов, который позволяет тестировщикам находить дефекты API, UI и логики в контролируемой среде.

---

## 🛠 Технологический стек

* **Бэкенд:** Node.js (Express) + База данных SQLite
* **Фронтенд:** React + TypeScript + Tailwind CSS
* **Инструменты тестирования:** Charles Proxy, Chrome DevTools, Python (автотесты)

---

## 📂 Структура проекта

* `backend/` — Серверная часть приложения, эндпоинты API и база данных.
* `frontend/` — Клиентская часть приложения (интерфейс таск-менеджера).
